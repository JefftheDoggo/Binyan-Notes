## Trig Identities

- Addition/subtraction
	- Sin
		- $sin(a\pm b) = sin(a)cos(b) \pm sin(b)cos(a)$
	- Cos
		- $cos(a\pm b) = cos(a)cos(b) \mp sin(a)sin(b)$
	- Tan
		- $tan(a\pm b) = \frac {tan(a)\pm tan(b)}{1\mp tan(a)tan(b)}$
- Double angle
	- Sin
		- $sin(2a) = 2sin(a)cos(a)$
	- Cos
		- $cos(2a) = cos^2(a)-sin^2(a)$
			- $\ \ \ \ \ = 2cos^2(a)-1$
			- $\ \ \ \ \ = 1-2sin^2(a)$
	- Tan
		- $tan(2a) = \frac{2tan(a)}{1-tan^2(a)}$
